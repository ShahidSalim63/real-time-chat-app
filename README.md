# Real-Time Chat Application

## Overview
A full-stack chat application that enables real-time communication using WebSockets.  
It is built with modern web technologies to support authentication, messaging, and a responsive UI.

## Features
- Real-time messaging with Socket.io
- User authentication with JWT tokens and OAuth
- Middleware-protected Express API routes
- Database models for users and messages using MongoDB with Mongoose
- Global state management with Zustand
- Responsive UI built with React, Tailwind CSS, DaisyUI, and Lucide React icons
- Support for multiple themes

## Tech Stack
- **Frontend:** React, Tailwind CSS, DaisyUI, Lucide React
- **Backend:** Node.js, Express, Socket.io
- **Database:** MongoDB (Mongoose)
- **Authentication:** JWT, OAuth
- **Other Tools:** Axios, Zustand
